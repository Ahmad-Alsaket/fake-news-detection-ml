# Dataset

This project uses the **Fake and Real News Dataset** available on Kaggle.

You can download it here:

https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

After downloading, place the following files in the project root directory:

```
Fake.csv
True.csv
```

The dataset is not included in this repository because it exceeds GitHub's file size limits.
